# Audiobook Creator

Convert PDF books and documents into streaming audiobooks with a web-based chapter player.

## Features

- Extracts text from PDF files using pdfplumber
- Splits books into chapters automatically
- Generates natural-sounding audio using Microsoft edge-tts
- Web-based streaming player with chapter navigation, speed control, and progress tracking
- Handles large chapters by automatic text chunking

## Install on HappyCapy

    /install https://github.com/donatius777/audiobook-creator

## Usage

After installing, simply upload a PDF and ask:

> "Convert this PDF into an audiobook"

The skill will extract text, detect chapters, generate audio, and launch a streaming player.

## Available Voices

- en-US-GuyNeural (male, default)
- en-US-JennyNeural (female)
- en-GB-RyanNeural (British male)
- en-US-AriaNeural (female, clear)
